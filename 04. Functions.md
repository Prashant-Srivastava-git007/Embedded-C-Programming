## ğŸ”¹ What is a Function?

A **function** is a block of code that performs a specific task.  
Instead of writing the same code repeatedly, we can define it once as a function and call it whenever needed.

### Example:
```c
#include <stdio.h>

void sayHello() {
    printf("Hello!\n");
}

int main() {
    sayHello(); // calling the function
    return 0;
}
```

## ğŸ”¹ Types of Functions

### 1. User-Defined Functions

Functions written by the programmer for specific tasks.

```c

    printf("Welcome!\n");
}

int main() {
    greet(); // calling user-defined function
    return 0;
}
```

### 2. Predefined (Library) Functions

These come built-in with C, such as `printf()` and `scanf()`.

```c

    printf("This is a predefined function.\n");
    return 0;
}
```
