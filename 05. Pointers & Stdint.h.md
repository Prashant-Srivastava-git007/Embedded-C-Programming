Pointers let you read and modify values directly using memory addresses.

    printf("After write: %d\n", ch);

    return 0;

## ğŸ” Pointer Increment & Decrement

Pointers can move forward or backward through memory based on their data type size.

```c
#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};
    int *p = arr;

    printf("Value at arr[0]: %d\n", *p);
    p++;   // move to next integer (4 bytes ahead)
    printf("Value at arr[1]: %d\n", *p);
    p--;   // move back to previous
    printf("Value back at arr[0]: %d\n", *p);

    return 0;
}
```

### ğŸ§¾ Output :
```
Value at arr[0]: 10
Value at arr[1]: 20
Value back at arr[0]: 10
```

ğŸ§  Note: For `int *`, pointer moves by 4 bytes; for `char *`, moves by 1 byte.
