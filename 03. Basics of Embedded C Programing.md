## ðŸ”¢ 2. Address of a Variable

* Every variable in C has a unique memory address.
* To get it, use the `&` (address-of) operator.

```c
#include <stdio.h>

int main() {
    int a = 5;
    printf("Address of a: %p\n", &a);
    return 0;
}
```

ðŸ§  Notes:

* %p â†’ prints address in hexadecimal
* &a â†’ gives memory address of variable a

## ðŸ”„ 2. Typecasting an Address

* To store the address as an integer value, you can typecast it.

```c
#include <stdio.h>

int main() {
    int a = 5;
    unsigned long long int addr = (unsigned long long int)&a;
    printf("Address as pointer: %p\n", &a);
    printf("Address as integer: %llu\n", addr);
    return 0;
}
```

## ðŸ§± 3. Storage Classes in C

Storage classes define:
  * Scope (visibility of a variable)
  * Lifetime (how long it stays in memory)

| Storage Class | Keyword    | Lifetime         | Scope                 | Description                                     |
| ------------- | ---------- | ---------------- | --------------------- | ----------------------------------------------- |
| Automatic     | `auto`     | Local            | Within function       | Default for local variables                     |
| Register      | `register` | Local            | Within function       | Suggests CPU register storage                   |
| Static        | `static`   | Program lifetime | Local to function     | Retains value between calls                     |
| External      | `extern`   | Program lifetime | Global (across files) | Used to share variables/functions between files |
